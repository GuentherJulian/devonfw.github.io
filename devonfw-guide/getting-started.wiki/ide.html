The devon IDE

First we have to clarify what we are talking about, when we mention the devon IDE:

The devon IDE is not one monolithic program that is installed with a traditional executable; rather it’s a collection of scripts which are invoked via command line to automate several, repetetive development tasks. These scripts then interact with other tools (e.g. Maven, Jenkins), frameworks (Spring, Angular) or third party IDEs (Eclipse, VS Code, IntelliJ IDEA) to streamline the development workflow.
As such, the package size of the IDE is initally very small, the setup is simple, the included software is portable and the whole environment is unobtrusive and light on system resources.
