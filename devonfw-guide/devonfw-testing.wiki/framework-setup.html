Framework Setup

This page will deal with the installation and setup process of the Selenium Testing Framework.

Requirements

  1. JDK 1.8

    An instance of the Java Development Kit (JDK) is required to use the Framework. Download and execute the JDK 1.8 installer from here.

    After the installation process, two environment variables have to be added:

    Variable Name Variable Value

    JAVA_HOME

    C:\Path\To\JDK

    PATH

    C:\Path\To\JDK\bin

    The following image shows exemplary values for the Environment Variables.

    The installation can be verified by executing the following command in the command line:

    java --version

  2. 7-Zip

    7-Zip is a tool being used to unzip the downloaded framework. It can be downloaded here.

Installation

As soon as all requirements are met, the framework can be installed.

  1. Downloading the Framework

    The framework can be downloaded here.

  2. Unzipping

    Use 7-Zip to extract the contents from the downloaded file to your desired location.

Starting the IDE

  1. Use the script "start-eclipse.bat" to run the Eclipse IDE.

  2. When running the IDE for the first time, a project called "allure-app-under-test" should already be imported as default. If not however, it can be re-imported manually.

  3. Click on File → Import → "Existing Maven Projects"

  4. When being prompted for the Project "Root Folder", select "allure-app-under-test". It’s located at:

    your_extracted_framework/workspace/allure-app-under-test

    Finally, select check the pom.xml and click on finish to import the project.

After the project was successfully imported, you should be greeted with the familiar Eclipse interface.
